⚡ frontend/ — Deploy on VERCEL
🔧
vercel.json
frontend/vercel.json
Tells Vercel to redirect all URLs to index.html. WITHOUT this, refreshing any page like /products gives a 404 error.
🌍
.env.production
frontend/.env.production
Vercel reads this automatically when building. Contains your Render backend URL. You do NOT need to set env vars on Vercel manually because of this file.
⚙️
vite.config.js
frontend/vite.config.js
Vite build settings. Proxies /api calls to localhost:5000 when running locally.
🔌
src/utils/api.js
frontend/src/utils/api.js
Axios instance. Has 35s timeout for Render cold starts. Automatically attaches JWT token to every request.
🧠
src/context/
frontend/src/context/
AuthContext, CartContext, WishlistContext — global state management using React Context API + hooks.
📄
src/pages/
frontend/src/pages/
Home, Products, ProductDetail, Cart, Checkout, Orders, Wishlist, Profile, Login, Register + admin pages.
🚀 backend/ — Deploy on RENDER
🔧
render.yaml
backend/render.yaml
Tells Render how to build and start the server. Sets buildCommand: npm install and startCommand: node server.js automatically.
🌍
.env
backend/.env
Local environment variables. NEVER push this to GitHub. On Render you add these manually in the Environment tab.
🖥️
server.js
backend/server.js
Main Express server. Sets up CORS, routes, middleware. Listens on 0.0.0.0 (required for Render).
🗄️
config/db.js
backend/config/db.js
Connects to MongoDB Atlas. Auto-retries every 5s if connection fails instead of crashing the server.
🔒
middleware/authMiddleware.js
backend/middleware/authMiddleware.js
protect() verifies JWT token. adminOnly() blocks non-admin users. generateToken() creates JWT on login.
🌱
config/seed.js
backend/config/seed.js
Run once to fill MongoDB with 12 products + 2 demo users. Run: npm run seed from inside backend folder.
🚀 Deployment Steps
▲  VERCEL — Frontend
1
Go to vercel.com → sign in with GitHub
2
Click "Add New" → "Project" → import your repo
3
Set Root Directory to: frontend
Framework: Vite  |  Build: npm run build  |  Output: dist
4
Add Environment Variable on Vercel:
KeyValue
VITE_API_URLhttps://shopverse-api-k4qj.onrender.com/api
5
Click Deploy — done ✅
The vercel.json in the frontend folder is already there — Vercel reads it automatically.
